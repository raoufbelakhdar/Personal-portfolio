UI ELEMENTS STRUCTURE

Go to the layers panel and you'll see a list of pages, named after each element. "Components", "Navbar", "Hero"... ext.
You can explore all the ui elements by navigating between the pages.

TYPOGRAPHY AND COLOR PALETTE

If you want to change the Font Style or the Color of any Element, you can edit it from the Local Style list in the right sidebar of your Editor.

ADD YOUR LOGO

To add your logo, go to the Components page where you'll find all the components.
Select the logo component and replace the current image with your logo. It will change across all the ui elements with the logo component.
